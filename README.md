## INSTALLATION

```bash
# USING NPM
$ npm i

#or
$ npm i --legacy-peer-deps

# USING YARN
$ yarn install
```

## Running the app

```bash
# USING NPM

$ npm run dev

# USING YARN

$ yarn dev
```
